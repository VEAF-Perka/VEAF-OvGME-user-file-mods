Sometimes it's really hard to spot contacts fr om far, especially on smaller monitors. This mod uses DCS World label system in attempt to realistically enhance contact spotting.
Mod adds dark grey, semi transparent dot over air contacts, which subtly increases number of visible pixels taken by contact (1.5-2 times more pixels), especially at longer distances.
For planes - starting from ~0.45nm (barely visible difference) up to ~10.5nm (where difference is really visible).
For missiles - starting from ~0.2nm up to 1.7nm.
-> Use "Dot only" setting in game 

Mod de libellé - ajoute un libellé (point semi-transparent) sur les avions qui évolue en fonction de leur distance - pour aider à les voir.
-> Utiliser le réglage "Dot only" (point seulement)
