Adds an in-game scratchpad where you can write down notes (open and close it with  LCtrl-LShift-X, release focus with ESC)
Use the "+L/L" button to add the coordinates of all the map markers (only when playing solo)
The scratchpad is saved everytime you enter text to the 0000.txt file in your Saved Games\DCS\Scratchpad folder


Ajoute un bloc-notes dans le jeu pour prendre des notes (ouvrir et fermer avec LCtrl-LShift-X, appuyer sur ECHAP pour libérer le focus)
Utilisez le bouton "+L/L" pour ajouter au bloc-note les coordonnées de tous les marqueurs sur la carte (seulement en solo)
Le bloc-note est sauvegardé dès que vous entrez du texte dans le fichier 0000.txt du répertoire Parties Sauvegardées\DCS\Scratchpad

-- Tested with DCS World 2.5.6.45915 --
